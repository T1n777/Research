# Research

This repository is a collection of notes, experiments, references, and random findings from topics I explore while researching different subjects.

## Purpose

The goal of this repo is to:
- Store research notes and observations
- Save useful resources and references
- Document experiments or small tests
- Keep track of ideas worth revisiting later

## Structure

The repository may include:
- Topic-specific folders
- Markdown notes
- Code snippets
- Reference links
- Small experiments

There is no strict structure — things are organized as needed while researching.

## Disclaimer

Most content here is raw research material and may be incomplete, messy, or experimental.
